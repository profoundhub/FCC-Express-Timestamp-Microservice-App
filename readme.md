### Steps:

# Installing Express

- $ npm install express --save

## Other Dependencies

- $ npm install body-parser --save
- $ npm install cookie-parser --save
- $ npm install multer --save

## Install Moment.js

- $ npm install moment --save

## Start Server:

- $ node server.js

- Visit it: http://localhost:8081/ OR http://127.0.0.1:8081
